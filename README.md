# dot-files
A collection of my personal dotfiles taken from my .config folder. This setup is installed on an Arch-based Linux system.

## Toolchain

 The following utilities are expected to be installed:

### Python
- **pipx**
  - `virtualenv`
  - `tox`
  - `ipython`
  - `ipdb`
  - `pyinstaller`
  - `poetry`
  - `uv`
- **pyenv**

### Rust
- **cargo**

### Terraform
- **tfenv** — manage and install Terraform versions

### Command-line Utilities
- **mosh** — SSH replacement  
- **nethogs** — network monitoring  
- **exa** — modern `ls` alternative  
- **bat** (batcat) — modern `cat` alternative  
- **nvtop** — GPU monitoring TUI  
- **fastfetch** — system info  
- **direnv** — automatic environment variable loading/unloading  

### Embedded Systems Utilities
- GCC ARM cross-compiler  
- STM32 Cube Programmer  

### Prompt
- **starship**

### Terminal Emulators
- **kitty**  
- **yakuake** — drop-down terminal

### Other
- **Rofi**
